# JPetStoreAutomation
Selenium automation project
This project automates key user journeys in the JPetStore demo application, including:

- User Registration (with dynamic data)
- Writing registered users into Excel/CSV
- Login Test using Excel/CSV (Data-Driven
- Extent Reports

1. Install Java 17
2. Install IntelliJ
3. Open the project
4. Let Maven download dependencies
5. Run testng.xml
